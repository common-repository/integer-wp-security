WP security is a plugin responsible by security with short steps and fix the mainly security problems.

== Description ==

Reduce risk for you site, running the easy hardening protection of mainly attacks.
With hardening you can control bad request, blocking ips with malicious action like comments spam, brute force login and others.

= Hardening: =

*   Disable Plugin Theme Editor
*   Hidden WordPress version
*   Deny WP Debug mode
*   Disable directory browsing
*   Disable author page
*   Fix corrects permissions
*   Change the prefix tables
*   Change the login url
*   2 Factor authentication
*   Google reCaptcha Option

= Ip block Activity Options  =

*   Blocks comments with spam
*   Access malicious url
*   Blocks brute force

= Mail notifications =

= Verifications: =

*   Prefix tables
*   Easy usernames
*   Easy passwords
*   Check PHP Version
*   Check wrong permissions

= Logs: =

*   Save errors
*   User managers
*   Plugin managers

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-sec.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
